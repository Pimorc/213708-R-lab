R Lab instructions for a course in experimental design and data analyses for ecological and environmental studies (ES 213708) 
at Chiang Mai University

A hands-on guide for learning and applying R programming in data analyses for ecological and environmental research studies. 
This manual is designed for students, researchers, and data enthusiasts who want to build up skills in data analysis, 
visualization, and statistical computing using R.

📚 Overview
This repository contains:

1.	Instructions for R programming and exercises.
2.	Example datasets for practice.
3.	R code for guiding learning.
4.	An exercise at the end of each lab session

🛠 Requirements
Before you begin, ensure you have:

1. R (version 4.0 or higher)
2. RStudio (recommended)
3. Required R packages (installations are included in each lab)

🚀 Getting Started

1.	Clone the repository
2.	Open in RStudio and set the working directory to the project folder.
3.	Start with Lab 1.

📖 Topics Covered
The lessons include:-

1.	Lab 1 – R download and basic.
2.	Lab 2 - data import and management
3.	Lab 3 – R data types and working with data
4.	Lab 4 – A normal distribution
5.	Lab 5 – Some probability and stochastic distributions	
6.	Lab 6 - Exploratory data analysis 1
7.	Lab 7 - Exploratory data analysis 2
8.	Lab 8 - Design of experiments (This lab is not using R)
9.	Lab 9 - Statistical modeling
10.	Lab 10 – A linear regression	
11.	Lab 11 – A generalized linear model
12.	Lab 12 - Analysis of Variance and Analysis of Covariance
13.	Lab 13 – A mixed model
14.	Lab 14 – Principal component analysis
15.	Lab 15 – Linear discriminant analysis
